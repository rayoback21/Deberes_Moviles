# ¿Qué es Zustand?

* a) Una biblioteca de gestión de estado para Angular.
* b) Una biblioteca de gestión de estado para React.
* c) Una biblioteca de gestión de estado para Vue.js.

Answer: c)

# ¿Cómo se puede definir y actualizar el estado en una aplicación que utiliza Zustand?

* a) Utilizando componentes de clase.
* b) Utilizando hooks de React.
* c) Utilizando directivas de Vue.js.

Answer: a)

* a) Su complejidad y robustez.

* c) Su integración exclusiva con jQuery.
